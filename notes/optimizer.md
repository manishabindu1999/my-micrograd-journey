# Optimizers

## What is an optimizer?

An optimizer updates neural network parameters using gradients.

Its goal is to minimize loss efficiently.

---

## Core Responsibilities

Optimizers:

- use gradients
- update parameters
- improve training
- stabilize optimization

---

## Common Optimizers

Examples:

- SGD
- Adam
- RMSProp

---

## Why Optimizers Matter

Different optimizers affect:

- training speed
- stability
- convergence quality

Modern deep learning heavily depends on good optimization algorithms.

Understand how optimizers improve neural network training.
