# Tiny Dataset Training

## What is tiny dataset training?

Tiny dataset training means training a neural network on a very small dataset.

This helps understand learning mechanics clearly.

---

## Why Tiny Datasets Matter

Small datasets make it easier to observe:

- forward propagation
- loss reduction
- gradient updates
- optimization dynamics

---

## Example Dataset

Inputs:

x = [2, 3, -1]

Target:

y = 1

The network learns to map inputs to targets.

---

## Training Pipeline

1. forward pass
2. compute loss
3. backward pass
4. update parameters
5. repeat

---

## Why This Is Important

Tiny experiments help build intuition for:

- neural network learning
- optimization
- parameter updates

Understanding small systems deeply is essential before scaling up.

---

## Goal

Train a tiny neural network from scratch using autodiff.
