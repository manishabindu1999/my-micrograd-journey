# Parameter Updates

## What are parameter updates?

Parameter updates modify neural network weights using gradients.

This is how neural networks learn.

---

## Core Equation

new_weight = old_weight - learning_rate * gradient

---

## Why Updates Matter

Gradients measure:

"How much does this parameter affect loss?"

Updates move parameters toward lower loss.

---

## Learning Process

Repeated updates gradually improve model predictions.

The network learns useful patterns from data.

---

## Types of Parameters

Neural networks update:

- weights
- biases
- embeddings
- normalization parameters

---

## Goal

Understand how gradients modify neural network parameters.
